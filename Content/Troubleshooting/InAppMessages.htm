In app messages

De-link component. This will de-link this component. If this component is registered it will be deregistered.

This message appears when you tap the recycle bin icon on a component of an equipment, such as the console, base, or network card. De-linking is necessary, for example, when a console is replaced with another console: first de-link the existing console and then add the new console.

Equipment only displays in a PBS location page when all three components are present (console, base, and network card).

 

Error, something unexpected went wrong

This occurs when an unexpected action is performed within the PrevaSync app. Touch OK to close the message. Contact Precor support with the steps to reproduce the issue and troubleshoot a solution.

 

Not found. Could not find equipment with a serial number of {notFoundSerialNumber} registered to this location.

This message appears when the scanned equipment is not found in PBS. Touch OK to close the message. To resolve this issue contact customer support.

 

Registered Elsewhere. The equipment with serial number ${registeredOtherSerialNumber} is registered to a different location.

Touch OK to close the message. Contact Precor support to resolve this issue.

 

Invalid trailing spaces

Network SSID cannot have untrimmed Spaces. Extra spaces can be accidentally added into fields when using the Paste feature. Remove any extra spaces in fields. Touch Cancel to make required changes or touch OK to close without saving any changes.

 

Model not found. This linkage contains an invalid combination of components. Please de-link incompatible components. Touch CANCEL to correct the problem or touch OK to continue without registering.

This message appears when you match incompatible devices together, such as an EFX console with a Treadmill. Touch Cancel and scan the correct equipment component.
