Add network

To network the equipment, you must first add the facility location and then add the location network in the PrevaSync app. Once the network is added, you can register the equipment.

To add location:

  1. Log on to the PrevaSync app. The home page appears.
  2. Touch the Register Equipment tile. A list of all locations accessible to the service technician appears.
  3. If your location does not appear, use the Search feature to find your location by the Location name (Facility name ex: ABC Gym) or the Location code.

If you are unable to find the location, ensure the IT/System Admin of the facility has added you to their Preva Business Suite as a Service Technician.

  1. Select the location. A list of networked equipment may appear. If no equipment is networked at the location, the page will be blank.

To add the network:

  1. Touch the Register Equipment tile.
  2. Select your facility location. The location settings page appears.
  3. Touch the Settings gear. The settings page appears.
  4. In the WiFi networks section, tap + Add Network.


  5. Provide the facility's Network name, Security, and Password.

Network name is case-sensitive.
  1. Set as default network toggle is auto set as default.
  2. Tap Save.
